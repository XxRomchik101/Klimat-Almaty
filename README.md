# Klimat Almaty

Сайт компании Klimat Almaty: установка и чистка кондиционеров в Алматы.  
Адаптивный дизайн, анимации при скролле, кнопки для звонка и WhatsApp, отзывы клиентов.  

## Доступно онлайн через GitHub Pages
Просто загрузите этот репозиторий и включите GitHub Pages в настройках репозитория.  

### Используемые технологии:
- HTML5  
- Tailwind CSS  
- Адаптивный дизайн для мобильных и ПК
